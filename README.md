# Dutch College CounterStrike Series Website

Personal Project. 
Designed and developed a full-stack web platform for an active esports' series, delivering player registration, team profiles, player profiles, and admin interfaces for both website registration check and tournament participation check.

Key highlights:
* Built with ASP.NET Core Razor Pages and a MySQL data layer for secure, efficient information management.
* Integrated Discord OAuth authentication to streamline player onboarding.
* Architected for scalability, supporting multiple seasons and diverse tournament formats.
* Operated successfully in a production environment for more than a year, proving reliability and maintainability.
